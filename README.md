# محفّظ القرآن الكريم بالذكاء الاصطناعي — النموذج الأولي

تطبيق ويب يساعد على تثبيت الحفظ وتصحيح التلاوة: يستمع إليك وأنت تتلو،
ويتعرّف على صوتك، ويقارنه لحظةً بلحظة مع النص القرآني، ويُبرز موضعك على الشاشة،
ويلوّن الكلمة بالأحمر عند الاختلاف.

هذا النموذج الأولي مطبَّق على **سورة الفاتحة** (رواية حفص)، وبنيته قابلة لتوسعة
بقية السور بالطريقة نفسها.

> هذه **المرحلة 1** من خارطة الطريق في وثيقة المشروع: تتبّع لحظي، محاذاة مرنة،
> وتلوين الأخطاء.

---

## التشغيل

واجهة التعرّف على الكلام والميكروفون تتطلّبان «سياقاً آمناً» (https أو
`localhost`)، لذا شغّل عبر الخادم المحلي بدل فتح الملف مباشرةً:

```bash
node server.js        # أو:  npm start
# ثم افتح:  http://localhost:8080
```

- اضغط **ابدأ الاستماع** وامنح إذن الميكروفون، ثم ابدأ التلاوة.
- لا تحتاج ميكروفوناً للتجربة: افتح **وضع الاختبار اليدوي** واكتب الكلمات.
- يعمل التعرّف الصوتي أفضل في متصفّح **Chrome**.

## الاختبارات

اختبارات وحدة لطبقتَي التطبيع والمحاذاة، تعمل عبر Node بلا أي تبعيات:

```bash
node test/aligner.test.js     # أو:  npm test
```

## كيف يعمل

ثلاث طبقات مفصولة، كما في وثيقة المشروع:

1. **طبقة الصوت (ASR)** — `js/speech.js`
   غلاف رفيع حول Web Speech API يحوّل التلاوة إلى كلمات. معزول عن بقيّة
   التطبيق ليسهل لاحقاً استبداله بنموذج محلي (whisper.cpp / Vosk) دون المساس
   بمنطق المحاذاة.

2. **طبقة المحاذاة والتصحيح (الجوهر)** — `js/normalize.js` + `js/aligner.js`
   مستقلّة تماماً عن محرّك الصوت:
   - **التطبيع:** إزالة التشكيل والتطويل، وتوحيد الحروف المتقاربة
     (أ/إ/آ/ٱ ← ا ، ى ← ي ، ة ← ه …) لتفادي الأخطاء الكاذبة.
   - **المحاذاة:** لكل كلمة مسموعة نبحث في نافذة حول «المرساة»
     (الكلمة المتوقَّعة التالية) بمدى `[المرساة − 4 ، المرساة + 6]`، ونحسب
     تشابهاً عبر مسافة ليفنشتاين مع ترجيح القرب. إذا تجاوز أفضل تشابه
     **العتبة** (افتراضياً `0.62`) نثبّت الكلمة صحيحةً وننقل المرساة؛ وإلا
     نعلّم الموضع المتوقَّع خطأً مؤقتاً ونُبقي المرساة لإتاحة إعادة النطق.
     النافذة هي ما يسمح بالقفز للأمام والرجوع للخلف دون فقدان الموضع.

3. **طبقة العرض** — `index.html` + `css/styles.css` + `js/app.js`
   عرض من اليمين لليسار بخط مصحفي، وتلوين الكلمات: **أخضر** صحيحة،
   **ذهبي** الموضع الحالي، **أحمر** اختلاف، مع شريط تقدّم وإشعارات.

## بنية المشروع

```
index.html            الواجهة
css/styles.css         التنسيقات (RTL، خط مصحفي، ألوان الحالات)
js/quran-data.js       نص سورة الفاتحة (قابل لإضافة سور أخرى)
js/normalize.js        تطبيع النص العربي
js/aligner.js          محرّك المحاذاة (ليفنشتاين + نافذة بحث + عتبة)
js/speech.js           غلاف Web Speech API
js/app.js              ربط المكوّنات والعرض
test/aligner.test.js   اختبارات وحدة (Node، بلا تبعيات)
server.js              خادم تطوير محلي بسيط
```

## الإعدادات

شريط **عتبة القبول** يضبط صرامة المطابقة لحظياً: أعلى = أدقّ، أدنى = أكثر
تسامحاً.
